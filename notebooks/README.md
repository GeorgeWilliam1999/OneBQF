# Notebooks

This directory contains Jupyter notebooks for experiments, analysis, and visualizations.

## Contents

- Experimental notebooks
- Data analysis and visualization
- Performance comparisons
- Example usage demonstrations

## Usage

1. Install Jupyter: `pip install jupyter`
2. Launch: `jupyter notebook`
3. Open and run notebooks

Add descriptive names to notebooks, e.g., `01_data_exploration.ipynb`, `02_ibqf_performance.ipynb`
